Downie——Mac必备视频下载神器

# 简介

Downie for Mac 是一款Mac OS平台上最好用的Mac视频下载工具，Downie for mac 破解版 支持1000多个不同的视频网站，支持国内的优酷、腾讯视频、bilibili、土豆、爱奇艺、网易、 搜狐、微博以及国外的Youtube等在线视频网站视频下载，功能强大使用简单。

# 用法

# 1.最简单最常用的用法

当你向你的朋友说借一部说话的时候，你的朋友向你丢出了如下链接

https://cdn.jsdelivr.net/gh/kimentanm/image-store/img/downie-test.mp4

当你将链接输入到浏览器地址栏并按下回车后，浏览器会将视频直接播放出来 如果你想视频下载下来，在浏览器中就无法直接操作了 此时你复制视频链接，打开Downie，在首页直接ctrl+v，视频就下载下来了

# 2.下载哔哩哔哩视频

找到你想要下载的视频，并复制浏览器中的链接，然后回到Downie，在首页ctrl+v复制链接

# 3.下载爱奇艺视频

与上面的操作类似，复制爱奇艺浏览器链接并在Downie中复制即可下载视频

注意

爱奇艺VIP视频也可以下载哦😎
